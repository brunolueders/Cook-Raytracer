# Cook Raytracer
### Summary
Distribution raytracer based on [Robert L. Cook's seminal 1984 paper](http://gec.di.uminho.pt/DISCIP/MCGAV/ifr0405/Artigos/Cook-DistributedRayTracing.pdf).

### Features
* Soft shadows
* Glossy reflections
* Translucency
* Depth of field
* Multithreaded rendering
